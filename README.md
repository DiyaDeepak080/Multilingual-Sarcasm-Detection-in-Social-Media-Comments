# Multilingual Sarcasm Detection in Social Media Comments

This project focuses on detecting sarcasm in social media comments using machine learning, deep learning, and transformer-based models. The study primarily targets Malayalam language comments collected from platforms such as YouTube and Reddit.

Sarcasm detection is a challenging natural language processing task because sarcasm often depends on contextual meaning, cultural references, and tone. To address this problem, multiple models were implemented and compared to determine which approach performs best for multilingual sarcasm detection.

## Models Implemented

The following models were implemented and evaluated:

### Machine Learning Models
- Logistic Regression
- Support Vector Machine (Linear & RBF)
- Random Forest

### Deep Learning Models
- Simple RNN
- CNN (1D Convolutional Neural Network)
- BiLSTM
- GRU

### Transformer Models
- Multilingual BERT (mBERT)
- XLM-RoBERTa

## Dataset

The dataset consists of Malayalam social media comments collected from:

- YouTube comments
- Reddit comments

The Reddit dataset was translated from English into Malayalam to simulate multilingual sarcasm detection scenarios.

## Features

- Text preprocessing and cleaning pipeline
- TF-IDF feature extraction
- Deep learning sequence models
- Transformer-based contextual models
- Performance comparison using Accuracy, Precision, Recall, and F1-score

## Objective

The goal of this project is to analyze how different machine learning and deep learning approaches perform in detecting sarcasm in multilingual social media text and to identify the most effective model for this task.
